# Notebooks

This directory contains exploratory analysis notebooks and development scripts for the financial complaint RAG chatbot project.

## Contents

- `task_1_eda_preprocessing.py`: Exploratory data analysis and preprocessing for Task 1
- `task_2_embedding_pipeline.py`: Embedding generation pipeline for Task 2
- `task_2_phase2_*.py`: Phase 2 embedding generation scripts
- `task_3_rag_with_prebuilt.py`: RAG implementation with prebuilt embeddings
